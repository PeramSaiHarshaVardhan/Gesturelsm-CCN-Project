# Gesturelsm-CCN-Project
This project is based on Gesture video which is generated by giving the audio.
Need to run every cell to get the final output. in middle you wil get some exception errors but the project will run at last. For this project kaggle environment is the best to get the output in a short period of time. because this is the big project it must be run in high GPU. kaggle environment will provide the more GPU than other.
